# Bite & Brew 🍽️

**Bite & Brew** is a modern restaurant web app built with React, designed to offer a sleek and responsive UI for browsing menus, placing orders, and more.


## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- Tailwind 
- React Router

## 📦 Installation

```bash
git clone https://github.com/priyacha123/Bite-Brew.git
cd Bite-Brew
npm install
npm run dev
